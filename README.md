## Идея компании

Анализируем статистику регулярного чемпионата Континентнаяльной Хоккейной Лиги (КХЛ) с целью выявленния причинно-следственных связей. Мы планируем создать сервис по анализу игровых показателей как игроков, так и команд с целью создания продажи аггрегированных данных командам (тренерам и скаутам), так и болельщикам и ставочникам. Возможно, мы будем также предлагать и рекоммендации на основе собранных данных.

### Примеры статистики:

1. Количество голов, бросков, силовых приёмов от каждого игрока каждой команды
2. Тепловая карта бросков каждого игрока.
3. Индивидуальные показатели вратарей.
4. Количество удалений при судействе конкретных арбитров.
